# PDI - Resumo Para Prova

Este repositório será usado para montar resumos em tópicos sobre Processamento Digital de Imagens, organizados por seção do conteúdo da prova.

## Como Vamos Trabalhar

- Você envia uma seção ou trecho do material.
- Eu transformo o conteúdo em resumo objetivo, em tópicos.
- Cada seção fica em um arquivo próprio dentro de `/docs`.
- Quando fizer sentido, adicionamos fórmulas, conceitos-chave, exemplos e pontos de atenção para prova.
